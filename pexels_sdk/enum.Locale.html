Expand description

指定搜索查询的语言环境。 支持的值：en-US、pt-BR、es-ES、ca-ES、de-DE、it-IT、fr-FR、sv-SE、id-ID、pl-PL、ja-JP、zh-TW、zh-CN、ko-KR、th-TH、nl-NL、hu-HU、vi-VN、cs-CZ、da-DK、fi-FI、uk-UA、el-GR、ro-RO、nb-NO、sk-SK、tr-TR、ru-RU。 默认值：en-US。

§示例

use pexels_sdk::Locale;
use std::str::FromStr;

let locale = Locale::from_str("en-US").unwrap();
assert_eq!(locale, Locale::en_US);
